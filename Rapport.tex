Notre projet consiste à créer une interface graphique de simulation de fluides en 2D. Cette
interface permet à l'utilisateur d'intéragir en modifiant les paramètres de la simuulation
afin d'observer l'évolution du fluide. Ce projet s'inscrit dans le cadre de notre licence en 
informatique avec l'UE Projet De Programmation 2 pour laquelle nous avons pu choisir un sujet 
parmi une liste donnée, et nous avons choisi celui-ci : la simulation de fluides en 2D.

{\color{red}A REDIGER (Saurane)

Notre interface graphique permet à l'utilisateur de vosualiser la simulation de fluides en 2D. 
L'interface permet de modifier les paramètres graphiques du fluide, tels que la densité, la vitesse, 
la direction, etc, mais aussi la position du fluide ainsi que le/les obstables présents et le mode 
de simulation choisi (point ou ligne).
Possibilité pour l'utilisateur de changer la forme de l'obstable. Par défault un cercle
est généré, on peut le déplacer avec la souris ou le supprimer avec le bouton "Obstable : ON/OFF", ou 
en appuyant sur O. On peut aussi changer la forme de cet obstable avec le bouton "Shape : Circle/Heart".
